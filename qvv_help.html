QVV Image Viewer Help

Images List Browser Help

For usual keyboard shortcuts check menus. Images list browser window handles the following keys:
[
show previous image
]
show next image
*
show random image
~
got to user home directory

Image View Help

Image view window handles the following keys:
[
show previous image
]
show next image
*
show random image
ARROWS
move image window
F3
show image list browser if hidden
F4
close image
ESC
close image
ENTER
close image
0
fit image to display if larger (scale down)
1
show image at 100% (1:1)
2
show image at 200%
3
show image at 300%
4
show image at 400%
5
show image at 500%
+
increase scale by 10%
-
decrease scale by 10%
z
scale image to screen size, keep aspect ratio
x
scale image to screen size
INSERT
center image

Browser/List Help

For usual keyboard shortcuts check menus. Browser (list) window handles the following keys:
[
show previous image
]
show next image
*
show random image
~
got to user home directory

Image View Help

Image view window handles the following keys:
[
show previous image
]
show next image
*
show random image
ARROWS
move image window
F4
close image
ESC
close image
ENTER
close image
0
fit image to display (scale down)
1
show image at 100% (1:1)
2
show image at 200% (1:1)
3
show image at 300% (1:1)
4
show image at 400% (1:1)
5
show image at 500% (1:1)
INSERT
center image
